Acerca del Barcamp

¿Qué es barcamp?

BarCamp es una red internacional de "desconferencias" tecnológicas, cuyo contenido es auto-generado por los participantes y su orientación es hacia el desarrollo de Software y tecnología en general.

¿Quién puede participar?

Cualquier desarrollador de software, profesores y estudiantes de distintas universidades pueden participar, ¡el único requerimiento es tener ganas de aprender más!

¿Cuándo?

BarCamp RD 2015 será celebrado en el campus de Santiago de la Pontificia Universidad Catolica Madre y Maestra, el sábado 28 de noviembre del 2015, desde las 09:00 A.M hasta las 6:00 P.M

Sesiones

Java 8 - Más allá de las Expresiones Lambda

Por: Eudris Cabrera

Java 8 es quizás el cambio más significativo en la historia del lenguaje, dado que incorpora varias novedades siguiendo la tendencia de otros lenguajes como la incorporación de lambdas o closures y una demandada nueva API para el manejo de fechas entre otras muchas características más. En esta charla vamos a mostrar a los interesados las nuevas características que ofrece Java SE 8, haciendo énfasis en las funcionalidades de programación funcional y los nuevo APIs.

Algebraic Data Types

Por: Rodolfo Hansen

Que conlleva la frase "Tipo de Dato Algebráico" y que consecuencias y utilidades nos ofrece empezar a manipular los tipos de datos algebráicamente.

De Programador a Arquitecto

Por: Alexander Matos

Interesados en aprender lo que el papel de un arquitecto de software conlleva y por qué es tan crucial para el éxito de cualquier organización.

Debugging para el no iniciado: De Frustrado a competente

Por: Leonardo Jiménez

Debugging no es un arte aunque lo parezca, es una ciencia, en esta charla te daremos una serie de tecnicas y metodos para hacer tu debugging mas placentero.

Django 101: Backend ágil con Django web Framework

Por: Ramón Emilio Ferreyra

Desconferencia introductoria al desarrollo web con Django framework, enfocada a dar a conocer sus virtudes a la hora de crear aplicaciones web de alto nivel y desempeño.

OpenCV en C#

Por: David de la Hoz

El reconocimiento de imágenes es el futuro de la computación y por que no hablar de ella realizando algunas demostraciones de sus usos y lo fácil que puede ser implementada usando C# bajo EmguCV

MongoDB con Estereoides

Por: Luis Contreras

Presentar base de Datos TokuMX y Percona Server para MongoDB y ver que ventajas tienen estas bases de datos sobre MongoDB, conoces caso puntuales en como pueden ser aplicados.

Big Data usando Postgres

Por: Edwin Quijada

Mostrar como poder usar PostgreSQL como RDBMS para el manejo e implementacion de Big Data.

El Movimiento Maker: Una nueva cultura de Innovación

Por: Roniel Guzmán

Conoce como el Movimiento Maker ha impactado en el desarrollo de nuevas tecnologías, creando una ola de innovación en todo el mundo y ya está en República Dominicana. Hablaremos sobre los temas mas interesantes en el Movimiento, figuras que pertenecen a el, como comenzar a ser parte de este movimiento y explicar el Zero To Maker.

A la garata con puños en algoritmia y programación

Por: Roberto Abreu

La programación competitiva es un deporte mental en el que (usualmente) miles de participantes compiten, intentando resolver la mayor cantidad de problemas fundamentalmente algorítmicos y de programación. Estaremos abordando el tema y narrando la historia de la República Dominicana en ella. Está dirigida principalmente a Profesores de Informática y Matemática, de cualquier nivel de enseñanza, así como a estudiantes de educación media y estudiantes de ciencias e ingeniería.

JAVA: Desarrollo de Aplicaciones Avanzadas, Eficiente y Mantenibles

Por: Clivens Petit

En esta Charla hablaremos de algunos temas avanzados de Java, como Interface, Herencia, Polimorfismo, Hilos optimizados (ExecutorService y Future), Sockets, Colecciones y Extensibilidad de programas a través de Plugins. Estaremos compartiendo buenas prácticas sobre cuando usar cada tecnología.

Integración de proyectos embebido Utilizando Arduino, Raspberry PI y Java

Por: Carlos Cmaacho & Javier Marte

En la charla estaremos mostrando los puntos importante a considerar en un proyecto de integración de componentes electrónicos para ser procesados por un sistema de alto nivel; abordando la elección del micro-procesador, protocolo de comunicación, técnicas de ahorro de energía y estrategias de integración con el sistema del alto nivel utilizando Arduino, Raspberry PI y la plataforma Java. En la presentación estaremos mostrando un proyecto basado en sistema embebido para procesar temperatura de ambiente y su integración en una aplicación web.
